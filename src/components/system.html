System

The system consists of three subsystems: a Hardware Sensor Platform, a Communication Interface, and a VR-Wearable.

Hardware Sensor Platform

Captures biometric and motion data through integrated sensors, designed for real-time streaming.

Communication Interface

Provides wireless BLE communication to relay data efficiently between hardware and host system.

VR-Wearable

An interactive VR model for data visualization and simulation within immersive training setups.

System Design

Simplified schematic representation of the simulation system. It is integrated into a VR system with tracking capabilities. On the left: the hardware sensor platform for capturing and processing sensor data. In the center: the communication interface, which acts as the master of the BLE connection and serves as a serial bridge. It enables wireless communication between the hardware sensor platform and the VR wearable. On the right: the VR wearable, which is integrated at the application layer into a Unity project.
