Communication Interface

Provides wireless BLE communication to relay data efficiently between hardware and host system.

The Arduino Nano 33 IoT sits at the heart of the communication interface. Powered via its onboard Micro-USB port (or by a 3.7 V LiPo cell), it acts as the I²C master for the SparkFun Bio Sensor Hub—polling heart-rate, SpO₂, and accelerometer data—then relays measurements either over USB serial or wirelessly via BLE to the host application.
