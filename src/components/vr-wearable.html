VR-Wearable

The VR wearable is integrated into Unity at the application layer and visualizes data directly within the scene.

To integrate with Unity, we use the official SteamVR Plugin from the Unity Asset Store. It provides all the necessary APIs for Valve Index and HTC Vive tracking, ensuring seamless positional tracking within the Unity scene. You can find it here: SteamVR Plugin (Asset Store) .

In addition, the VR-Wearable SDK provides all the building blocks for rendering and controlling SpiderClip in Unity. It includes C# scripts, prefabs, and demo scenes, and is available as an open-source project on GitHub: VR-Wearable SDK (GitHub) .
